# フェーズ4: 管理者用フロントエンド UIテスト実装計画

## 概要

管理者用フロントエンドシステムの包括的なUIテスト実装計画です。画面コンポーネントレベルのテストから、ユーザシナリオベースのテストまで、品質保証に必要な全てのテストケースを体系的に整理しています。

## テスト方針

### 基本方針
- **画面部品レベル**: 全UI要素の入出力・エラー表示を網羅的にテスト
- **シナリオベース**: ユーザ利用・運用保守の全シナリオをテスト
- **正常系・異常系**: 各テストケースで正常系・異常系を必ず含める
- **高負荷テスト除外**: 大量データ・API・同時操作は対象外

### テスト技術スタック
- **Selenium WebDriver**: ブラウザ自動化・E2E UIテスト
- **pytest**: テストフレームワーク・フィクスチャ管理
- **pytest-selenium**: Selenium統合・並列実行
- **pytest-html**: レポート生成・可視化
- **Faker**: テストデータ生成
- **Pillow**: 画像生成・検証

## UI画面構成

### 1. メインダッシュボード (`/`)
**概要**: システム全体の状態監視とナビゲーション

**画面コンポーネント**:
- ヘッダー: タイトル・ユーザー情報・ナビゲーションメニュー
- システム状態パネル: ドローン接続状態・バッテリー・通信状況
- 機能メニュー: ドローン制御・カメラ・モデル訓練・追跡制御へのリンク
- アラート・通知エリア: エラー・警告・情報メッセージ表示
- フッター: バージョン情報・ヘルプリンク

### 2. ドローン制御画面 (`/drone`)
**概要**: ドローン接続・飛行制御・移動制御

**画面コンポーネント**:
- 接続制御パネル: 接続・切断ボタン・接続状態表示
- 基本飛行制御: 離陸・着陸・ホバリング・緊急停止ボタン
- 移動制御: 方向移動ボタン(前後左右上下)・回転制御
- センサー情報表示: バッテリー・高度・温度・姿勢角のリアルタイム表示
- ログ表示エリア: コマンド実行履歴・エラーログ

### 3. カメラ・ストリーミング画面 (`/camera`)
**概要**: ライブ映像表示・撮影・録画機能

**画面コンポーネント**:
- ビデオストリーム表示: リアルタイム映像・フルスクリーン対応
- ストリーミング制御: 開始・停止・一時停止ボタン
- 撮影制御: 写真撮影・連写・録画開始/停止ボタン
- カメラ設定: 解像度・フレームレート・ビットレート調整
- プレビュー・履歴: 撮影画像・動画のサムネイル表示

### 4. モデル訓練画面 (`/model`)
**概要**: 物体認識モデルの訓練・管理

**画面コンポーネント**:
- 画像アップロードエリア: ドラッグ&ドロップ・ファイル選択・プレビュー
- 訓練設定フォーム: オブジェクト名・訓練パラメータ入力
- 訓練制御: 開始・停止・一時停止ボタン
- 進行状況表示: プログレスバー・ログ・エラー表示
- モデル管理: 訓練済みモデル一覧・削除・切り替え

### 5. 物体追跡画面 (`/tracking`)
**概要**: 物体追跡の制御・監視・可視化

**画面コンポーネント**:
- 追跡制御パネル: 対象選択・追跡モード・開始/停止
- 状態監視: 追跡状態・検出状況・位置情報のリアルタイム表示
- ビデオオーバーレイ: 検出枠・軌跡・中心マーカー表示
- 統計・ログ: 追跡時間・精度・履歴・エラーログ
- 設定: 追跡パラメータ・表示オプション調整

    @staticmethod
    def assert_error_message_displayed(driver, message_text):
        """エラーメッセージ表示確認"""
        error_elements = driver.find_elements(By.CLASS_NAME, "error-message")
        assert any(message_text in elem.text for elem in error_elements), \
            f"Expected error message '{message_text}' not found"
    
    @staticmethod
    def assert_loading_indicator_present(driver):
        """ローディングインジケーター確認"""
        loading = driver.find_element(By.CLASS_NAME, "loading-spinner")

    @staticmethod

        """フォームバリデーションエラー確認"""
