📖 使用方法

接続手順

  1. Tello EDU ドローンの電源を入れる
  2. PCをドローンのWiFiネットワークに接続
  3. 「接続」ボタンをクリック
  4. 接続状態が「接続中」になることを確認

注意事項

  • 接続中は他のTelloアプリを閉じてください
  • バッテリー残量が20%以下の場合は飛行できません
  • 通信が不安定な場合は再接続してください
  • 緊急時は物理的な電源ボタンで停止できます
