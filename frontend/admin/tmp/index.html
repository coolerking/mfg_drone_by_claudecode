機能概要

🎯 主要機能

  • • ドローン制御: 離陸・着陸・移動・回転
  • • 物体追跡: AI による自動追跡・撮影
  • • ライブ映像: リアルタイム映像ストリーミング
  • • モデル管理: 物体認識モデルの学習・管理

🔧 技術仕様

  • • Frontend: HTML5 + CSS3 + JavaScript
  • • Backend API: FastAPI + Python 3.11
  • • ドローン: Tello EDU (djitellopy)
  • • プラットフォーム: Raspberry Pi 5

使用方法

注意: これらはモックアップです。実際のドローンとの通信は行われません。

📱 操作方法

  • 1. 上部のナビゲーションから画面を選択
  • 2. 各画面で機能を確認
  • 3. ボタンクリックで操作感を体験
  • 4. レスポンシブデザインを確認

⌨️ キーボードショートカット

  • • C: ドローン接続
  • • D: ドローン切断
  • • T: 離陸
  • • L: 着陸
  • • 矢印キー: 移動
